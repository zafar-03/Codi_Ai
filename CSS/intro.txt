How to Impletement CSS with our HTML ?
1. Inline CSS :-
    - always written in a starting tag.
    - use style attribute

2. Internal CSS :-
    - use style tag in a HTML file.
    - must be written in a Head Tag.

3. External CSS :-
    - create a new file with '.css' extension.
    - link/connect css file with our HTML file. 


Note : 
    we can use HTML without CSS. 
    but we cann't use CSS without HTML.
